# Trabajos de Programación I - 1C 2026 UTN Delta

Repositorio con mis ejercicios, prácticas y trabajos realizados durante
la materia **Programación I (1er cuatrimestre 2026)**. Este repositorio
irá creciendo y actualizándose a medida que avance el curso.

Aquí encontrarás ejercicios realizados en **C++**, junto con algunos
algoritmos desarrollados en **PSeInt** utilizados durante el aprendizaje
inicial.
